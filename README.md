# Digital Clock using Python 🕐

My first project as a Python Intern at InternPe IT Services.
A real-time digital clock built using Python that displays 
live time on your desktop screen!

## 🛠️ Tools & Libraries Used
- Python 3.12
- tkinter — for building the desktop window
- strftime() — for reading and formatting live system time

## ⚙️ How It Works
- Fetches current time from your system
- Displays it in HH:MM:SS AM/PM format
- Refreshes every 1 second using .after(1000)
- Runs as a fully working desktop application

## ▶️ How to Run
- python digital_clock.py
